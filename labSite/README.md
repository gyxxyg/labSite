# labsite

> The website of professor Tang's laboratory.

## Install vue

``` bash
# install vue
npm install vue

#install vue-cli
npm install -g vue-cli

#install webpack
npm install -g webpack

# install dependences
cd labSite
npm install

#run the code
npm run dev
```

## config vscode

>You can also choose Webstorm to deploy the project.
